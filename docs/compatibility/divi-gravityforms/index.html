Community Q&A

Are Divi and Gravity Forms compatible?

Yes, they are compatible. You can display Gravity Forms in Divi by using the provided shortcode within a text or code module.

How do I add a Gravity Form to a Divi page?

Simply copy the shortcode from your Gravity Form settings and paste it into a Divi Text, Code, or Blurb module.
