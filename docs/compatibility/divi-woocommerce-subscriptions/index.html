Community Q&A

Does Divi work with WooCommerce Subscriptions?

Yes, Divi is compatible with WooCommerce Subscriptions, but you may need to adjust CSS or use Divi’s WooCommerce modules to ensure the subscription checkout flows look correct.

Are there known styling issues with Divi and Subscriptions?

Users often report that the default WooCommerce subscription tables or buttons may not inherit Divi’s global styles, requiring custom CSS to align them.
