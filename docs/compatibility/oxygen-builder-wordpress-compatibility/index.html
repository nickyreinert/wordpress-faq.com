Community Q&A

Why does my Oxygen Builder site look different on Safari?

Safari often interprets CSS flexbox properties differently than Chrome, frequently requiring explicit width or flex-basis values to render correctly.

How do I fix flex-shrink issues in Oxygen Builder?

Ensure you have set an explicit width or flex-basis on the child elements, as Safari may ignore flex-shrink if the parent container lacks defined constraints.
