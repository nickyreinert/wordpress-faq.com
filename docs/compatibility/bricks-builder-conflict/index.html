Community Q&A

Is Bricks Builder a plugin or a theme?

Bricks Builder is a WordPress theme, not a plugin, which is a common point of confusion for new users.

Can Bricks Builder cause conflicts with other themes?

Yes, because Bricks is a theme, activating it will replace your current theme, which may cause layout or functionality conflicts.
