Known Issues

  • Severe page speed and performance degradation
  • Shortcode clutter when disabling Divi
  • Layout and styling conflicts between builder CSS files

Community Q&A

Can I run both Divi and Elementor on the same WordPress site?

While technically possible, it is highly discouraged because loading two heavy page builder frameworks severely impacts site loading speeds and causes layout conflicts.

What happens to my content if I switch from Divi to Elementor?

Divi leaves behind a large amount of shortcode clutter when deactivated. You will need to manually rebuild your pages within Elementor to restore proper layout and styling.
