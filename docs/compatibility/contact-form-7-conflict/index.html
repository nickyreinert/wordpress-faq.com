Community Q&A

Why is my Contact Form 7 not sending emails?

Email delivery issues are often caused by your host blocking default PHP mail. Using an SMTP plugin to route emails through a dedicated provider usually resolves this.

Can I use Recaptcha v2 and v3 together in Contact Form 7?

No, using both versions simultaneously causes conflicts. You should choose one version and ensure all old integration keys are fully removed from the settings.
