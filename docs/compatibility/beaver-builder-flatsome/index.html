Community Q&A

Can I use Beaver Builder with Flatsome?

Yes, you can use them together, but it is not recommended as Flatsome has its own built-in UX Builder which may conflict with Beaver Builder.

Will using two page builders slow down my site?

Yes, loading multiple page builder frameworks simultaneously increases the number of scripts and styles, which negatively impacts page performance.
