Community Q&A

Does WPForms work with Beaver Builder?

Yes, WPForms is fully compatible with Beaver Builder and can be easily embedded using the WPForms module or shortcode.

How do I add a WPForms form to a Beaver Builder page?

You can use the dedicated WPForms widget within the Beaver Builder editor or paste the form shortcode into an HTML module.
