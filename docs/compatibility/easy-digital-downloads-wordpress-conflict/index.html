Community Q&A

How do I identify a conflict with Easy Digital Downloads?

Deactivate all plugins except Easy Digital Downloads and switch to a default WordPress theme to see if the issue persists.

Can a WordPress theme break Easy Digital Downloads?

Yes, themes that override EDD template files or include conflicting JavaScript can cause checkout or download errors.
