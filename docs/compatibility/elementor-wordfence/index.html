Known Issues

  • Wordfence firewall may occasionally block Elementor’s AJAX saving requests as false positives.
  • Admin actions in Elementor might trigger Wordfence security alerts if Learning Mode is disabled.

Community Q&A

Are Elementor and Wordfence compatible?

Yes, Elementor and Wordfence work well together. You may just need to configure Wordfence’s firewall to avoid blocking Elementor’s saving processes.

Why does Wordfence block Elementor from saving pages?

This happens when the Wordfence Web Application Firewall flags Elementor’s AJAX requests as suspicious. Whitelist these actions using Wordfence Learning Mode.
