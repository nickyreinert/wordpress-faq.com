Community Q&A

What is the Gutenberg block editor?

Gutenberg is the default block-based editor for WordPress, allowing users to build content layouts using individual blocks instead of a single text area.

How do I update Gutenberg features?

Gutenberg updates are bundled with WordPress core updates. Keep your WordPress installation updated to the latest version to access new features.
