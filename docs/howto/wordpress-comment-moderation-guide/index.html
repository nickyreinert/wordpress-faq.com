Community Q&A

How do I approve comments in WordPress?

Navigate to the Comments screen in your dashboard, hover over a comment, and click Approve.

How can I stop spam comments on my site?

Enable comment moderation settings in Discussion settings or install an anti-spam plugin like Akismet.

Can I disable comments on specific posts?

Yes, you can uncheck ‘Allow comments’ in the Discussion meta box on the individual post edit screen.
