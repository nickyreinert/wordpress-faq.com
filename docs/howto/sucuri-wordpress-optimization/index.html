Community Q&A

How do I optimize Sucuri for WordPress?

To optimize Sucuri, ensure the WAF is correctly routed via DNS, enable the caching features within the Sucuri dashboard, and whitelist your server IP.

Does Sucuri slow down WordPress?

Sucuri can impact speed if the WAF is not configured correctly; using their CDN and caching features usually results in a net performance gain.
