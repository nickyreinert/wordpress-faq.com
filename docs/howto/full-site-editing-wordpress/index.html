Community Q&A

What is Full Site Editing in WordPress?

Full Site Editing is a collection of features that allows users to edit every part of their site, including headers, footers, and templates, using blocks.

Do I need a special theme for Full Site Editing?

Yes, you need a block-based theme to utilize the full capabilities of the Site Editor in WordPress.
