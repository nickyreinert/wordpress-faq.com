Community Q&A

How do I install Bricks Builder?

Download the zip file from your account, go to Appearance > Themes in WordPress, click Add New, upload the file, and activate it.

Does Bricks Builder work with other themes?

Bricks is a theme itself, so it replaces your current theme. You should not run it alongside other themes.
