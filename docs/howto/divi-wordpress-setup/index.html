Community Q&A

How do I install the Divi theme in WordPress?

Download the Divi zip file from your Elegant Themes account, go to Appearance > Themes > Add New, upload the file, and click Install.

Do I need a child theme for Divi?

Yes, it is highly recommended to use a Divi child theme to preserve your custom CSS and PHP changes during theme updates.
