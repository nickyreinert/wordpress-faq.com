What are the first settings to change in WordPress?

You should immediately update your permalinks, site title, tagline, and timezone settings to ensure proper SEO and site functionality.

Should I delete default WordPress plugins?

Yes, removing unused default plugins like Hello Dolly or Akismet (if not needed) reduces your attack surface and improves site performance.
