Community Q&A

How do I install Ninja Forms on WordPress?

Navigate to Plugins > Add New, search for Ninja Forms, click Install Now, and then Activate.

Does Ninja Forms support conditional logic?

Yes, conditional logic is supported, though advanced features may require the Conditional Logic add-on.
