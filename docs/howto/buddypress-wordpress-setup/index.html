How do I install BuddyPress on WordPress?

Navigate to Plugins > Add New, search for BuddyPress, click Install Now, and then Activate the plugin.

Does BuddyPress work with all WordPress themes?

While it works with most, some themes require specific BuddyPress template files to display social features correctly.
