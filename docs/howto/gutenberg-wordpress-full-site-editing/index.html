Community Q&A

What is Full Site Editing in WordPress?

Full Site Editing (FSE) is a collection of features that allow users to edit their entire website, including headers, footers, and templates, using blocks.

Is Gutenberg the same as a page builder?

While Gutenberg is a block editor, it has evolved to include page builder-like functionality, allowing for complex layouts without external plugins.

How do I start using Full Site Editing?

You need a block-based theme and the latest version of WordPress to access the Site Editor under the Appearance menu.
