Community Q&A

How do I translate pages with Polylang in WordPress?

You can duplicate your original page into the target language using the Polylang interface, then manually update the content within the Gutenberg editor.

Does Polylang automatically sync changes to translated pages?

No, Polylang does not automatically sync content changes. You must manually update the translated pages after modifying the original source page.
