Community Q&A

Can I use Avada with other page builders?

It is not recommended to use Avada alongside other page builders like WPBakery, as this often leads to CSS conflicts and layout issues.

How do I install the Avada theme?

Upload the Avada zip file via Appearance > Themes > Add New, then activate the theme and run the setup wizard to install required plugins.
