Community Q&A

How do I install the Astra theme?

Navigate to Appearance > Themes > Add New, search for Astra, and click Install then Activate.

Is Astra compatible with page builders?

Yes, Astra is designed to work seamlessly with Elementor, Beaver Builder, and the WordPress block editor.
